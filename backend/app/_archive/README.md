# Archive — SMC Modules

These modules were part of the original Smart Money Concepts (SMC) implementation.
They have been archived here and removed from the active codebase as the project
direction shifted to a broader professional technical analysis platform.

They are preserved for future use when BOS/CHOCH and order block detection are
added as **optional confirmation modules**.

## Contents
- `events/bos.py` — Break of Structure detector
- `events/choch.py` — Change of Character detector
- `events/mss.py` — Market Structure Shift detector
- `events/detector.py` — Base event detector
- `zones/order_blocks.py` — Order block detection
- `zones/fvg.py` — Fair Value Gap detection
- `test_bos.py` — BOS unit tests
